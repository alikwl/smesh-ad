# Smesh.dev Admin Dashboard

This is the admin dashboard for **smesh.dev**. It is built using Astro and connects directly to Supabase.

## Setup

1. Copy `.env.example` to `.env` and fill in the values.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build the production site:
   ```bash
   npm run build
   ```
